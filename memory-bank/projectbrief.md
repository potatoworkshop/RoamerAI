# RoamerAI 项目简要介绍

## 核心目标
RoamerAI 旨在创建能够像人类一样自主漫游互联网的AI机器人。这些机器人通过浏览器自动化技术，在网络上自由浏览并收集信息，而非执行预定义的具体任务。

## 项目范围
- 开发核心AI漫游逻辑
- 实现浏览器自动化基础设施
- 建立信息收集和存储机制
- 设计自主决策系统

## 核心要求
1. AI机器人必须收集信息作为主要行为
2. 支持自主决策信息收集方式
3. 模拟人类般的互联网漫游模式
4. 非任务导向，强调探索性信息收集

## 成功标准
- 机器人能够成功导航和浏览各种网站
- 自主生成和执行信息收集策略
- 持续收集并存储有价值的信息
- 维护系统的稳定性和性能
